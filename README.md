# MSA Mathe Prüfungsapp

Interaktive Web-App zum Üben der Berliner eBBR/MSA-Mathematikprüfungen für die Jahrgänge 2025, 2024 und 2023.

Live-App: https://vincent808s.github.io/msa-mathe-pruefungsapp/

## Funktionen

- Prüfungsauswahl nach Jahrgang
- Antwortfelder direkt unter jeder Aufgabe
- Foto-Upload pro Aufgabe für handschriftliche Rechenwege oder Skizzen
- Fortschrittsanzeige
- Korrekturmodus nach Abgabe
- Abgleich von Zahlenantworten mit den Serlo-Musterlösungen
- lokale Speicherung der Antworten im Browser

## Nutzung

1. Live-App öffnen.
2. Jahrgang auswählen.
3. Prüfung starten.
4. Antworten eintragen oder Fotos anhängen.
5. Am Ende Prüfung abgeben und Lösungen vergleichen.
